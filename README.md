# National Club Management Dashboard

A clean, one-screen dashboard for analyzing club and player data.  
It includes KPI cards, filters, charts, a leaderboard, auto insights, Google Sheets CSV connection, dark/light mode, and export options.

## Features

- Google Sheets CSV connection
- Raw CSV data support
- Interactive filters
- KPI cards
- Charts and heatmap
- Top players leaderboard
- Auto insights
- Data quality checks
- Dark and light mode
- Export to CSV
- Print / save as PDF

## How to Use

1. Open the dashboard in your browser.
2. The dashboard will load sample data automatically.
3. To use your own data, click **Connect Sheet**.
4. Paste a published Google Sheets CSV link or paste raw CSV data.
5. Click **Load Club Data**.
6. Use the filters on the left side to analyze the data.
7. Use the export buttons to download filtered data or top players.

## Google Sheets Setup

To connect a Google Sheet:

1. Open your Google Sheet.
2. Go to **File → Share → Publish to web**.
3. Choose CSV format.
4. Copy the published CSV link.
5. Paste it into the dashboard using **Connect Sheet**.
